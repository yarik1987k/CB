== Installation ==
<h4> By downloading it from wordpress.org </h4>
<ol>
<li>Download the <strong>duplicate-post-page-menu-cpt.zip</strong> file and unzip it.</li>
<li>Upload the <strong>duplicate-post-page-menu-cpt</strong> folder to the wp-content/plugins/ folder.</li>
<li>Activate the <strong>Duplicate Post Page Menu & Custom Post Type</strong> from plugin page on wordpress admin area.</li>
</ol>

<h4>Alternatively, you can install the plugin automatically through the WordPress Admin Panel</h4>
<ol>
<li>Open your WordPress website admin panel and go to Plugins > Add New </li>
<li>Search by 'Author' for '<strong>inqsys</strong>'</li>
<li>Click on <strong>Install Now</strong> button and then click on <strong>Activate</strong> Plugin link</li>
</ol>

== Frequently Asked Questions ==
= Why do I need this plugin? =
<p>You can use this plugin to create a duplicate entry of a page, post, custom post type and menu and treat it as a backup entry. Even if you don't have much content on your site to check your website's layout, you can duplicate a single post for multiple times just to insure that the website's design is as you want.</p>
= What all I can duplicate using this plugin? =
<p>This plugin is helpful in duplicating page, post, custom post type and menu.</p>
= Can I duplicate my products, coupons, team member etc kind of post entries? =
<p>Yes, all such type of posts are known as custom post and you can also duplicate them using this plugin.</p>
= What if I don't want this plugin to duplicate my post? =
<p>You can easily disable/enable the duplication of post, page and menu by checking/unchecking the specific item in settings. After activating the plugin, visit: Settings > 'Duplicate' Option</p>
= How many post/page I can duplicate at a time using this plugin? =
<p>You can only duplicate one post/page/menu per operation. However, you can choose to create more than one duplicate entries of the same post/page/menu.</p>
